Projects

  • BambooOS

    BambooOS is a feature-rich, x86_64 Operating System written in C with a UEFI bootloader, command-line, and support for:

    • Higher-half kernel
    • ACPI tables (RSDT, XSDT, MCFG, MADT)
    • Symmetric Multiprocessing (starting all other APs)
    • Full physical memory manager
    • Virtual memory manager
    • Heap allocator (malloc, free, realloc)
    • PCI support
    • Standard library functions
    • Interrupt handling
    • R/W to disk and FAT32 driver

  • Engine Simulator

    Engine Simulator is an application designed to accurately simulate the sound of an engine. Written in C++, the application uses physics and fluid simulations to model the physical engine, combustions, valves, and much more. The application also comes with a custom, completely hand-written UI drawn pixel-by-pixel, which is beautifully animated.
